Cristata App & Server

This subscription comes with the following benefits:

  • Hosting of the Cristata Web App.
    • Profile management with user-editable fields
    • Content Management System (CMS) with unlimited entries and adjustable visibility/editing permissions
    • Photo library interface with support for tags and photo attribution
    • Reusable components for further customization and expansion of Cristata
  • Hosting of the server that powers the Cristata Web App.
  • Realtime collaboration between multiple writers in the text editor.
  • Configuration of custom API collections per your specification.
  • Special API endpoints for information to be exposed to your public-facing website.

The Cristata server requires your credientials for a MongoDB database and an Amazon S3 storage bucket.


At this point in time, Cristata is only available to organiations. Do not attempt to pay for Cristata without prior approval.
